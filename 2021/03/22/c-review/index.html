gcc 命令如果不指定目标文件名时默认生成的可执行文件名为 a.out(linux) 或 a.exe(windows)。

可用 gcc [源文件名] -o [目标文件名] 来指定目标文件路径及文件名。

因编译器的原因，生成的 .exe 文件打开时会一闪而过，从而观察不到其运行的结果，这是因为 main() 函数结束时，DOS窗口会自动关闭。为了避免这个问题可在 return 0; 前加入 system("pause"); 语句。
