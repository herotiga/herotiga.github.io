作用域

同一个css变量可以在多个选择器中被声明，按照css层叠来选择优先级最高的声明。

总结：css变量的作用域就是声明其的选择器的范围，并且按照层叠，优先级越高的作用越优先。

一般将css全局变量放在:root中，以使得所有选择器都可读取。
